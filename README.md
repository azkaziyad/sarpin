 # Sarpras Inventory App

Aplikasi inventaris barang sarpras (sarana dan prasarana) untuk mengelola inventaris sekolah. Aplikasi ini memudahkan pengelolaan data barang, peminjaman, dan monitoring inventaris secara digital.

## Fitur Lengkap Aplikasi

### 1. Landing Page
- Halaman awal dengan informasi singkat tentang aplikasi dan pengenalan fitur
- Desain responsif 
- Navigasi mudah ke halaman login

### 2. Sistem Autentikasi
- **Login**: Form username dan password 

### 3. Dashboard
- Ringkasan statistik inventaris (total barang, peminjaman aktif, dll.)
- Aksi cepat untuk navigasi ke fitur utama
- Daftar barang tersedia dengan nama, kategori, stok, dan deskripsi
- Desain responsif dan simple

### 4. Data Barang (CRUD)
- **Create**: Tambah barang baru dengan nama, kategori, jumlah, dan deskripsi
- **Read**: Lihat daftar semua barang
- **Update**: Edit informasi barang
- **Delete**: Hapus barang dari inventaris
- Data disimpan di localStorage browser

### 5. Riwayat Peminjaman
- Tabel daftar peminjaman barang
- Data dummy untuk demo
- Kolom: Nama Barang, Peminjam, Tanggal Pinjam, Tanggal Kembali, Status
- Filter dan Pencarian Barangg

### 6. About Aplikasi
- Deskripsi lengkap aplikasi
- Kontak dan informasi tambahan
- Fitur aplikasi

## Stack Teknologi

- **Framework**: Next.js (App Router)
- **Frontend**: React 
- **Bahasa**: TypeScript
- **Styling**: Tailwind CSS 
- **UI Components**: Shadcn UI
- **Font**: Geist Sans & Geist Mono
- **Data Storage**: localStorage
